Installation
------------
### Some requirements
 完全放弃折腾, 最多使用自带的vi, 考虑如何以最快的时间在一台主机上配置
 一个趁手的Emacs.

### Install Emacs

    wget http://ftp.gnu.org/gnu/emacs/emacs-24.3.tar.xz
    tar Jxf emacs-24.3.tar.xz
    ./configure --with-x-toolkit=no --with-xpm=no --with-jpeg=no with-png=no --with-gif=no --with-tiff=no
    make && make install

Blogging with jekyll


Install ruby


$ wget http://cache.ruby-lang.org/pub/ruby/2.1/ruby-2.1.1.tar.gz
$ tar -xf ruby-2.1.1.tar.gz
$ cd ruby-2.1.1
$ ./configure && make && make install


Install jekyll


$ gem install jekyll

Setup jekyll


$ jekyll new jekyll-init
$ cp -r jekyll-init/*  {username}.github.io

$ git add .
$ git commit -m "Life is good, we use jekyll now \!"
$ git push
